Fun with the Command Line

If you are a true fan of Star Wars you will find yourself 'reliving' your first viewing of Episode IV when you try the first experiment here.

I was there, in 1977, for the preview of (what was known only as "Star Wars" at the time). {I have a photo of me with my friends under the marquee at the Loews Astor Plaza in New York}

This is a collection of simple Linux CLI tricks and quirks. Although they may work equally well on any Linux distribution, I have tested on my Nitrous box with positive results. Another great reason for using Nitrous is that YOUR MILEAGE WILL CERTAINLY NOT VARY!

For all of these, open the Console to full-screen in Nitrous

The Original Hope

Get a bowl of popcorn and type this at the command prompt:

telnet towel.blinkenlights.nl

If you do not wish to watch The Entire Movie you can use Ctrl+] to break then type ‘quit`

ESREVER

At the command prompt type rev
Anything you type next will be reversed. Use Ctrl+c to stop it

[Re]Factor

At the command prompt type factor
Any number you type will be repeated with all its factors

Use Ctrl+c to stop it

Back To School

Enter this one-line script at the command prompt

for i in {1..12}; do for j in $(seq 1 $i); do echo -ne $i\*$j=$((i*j))\\t;done;echo;done

For 'old folks' like me, this immediately brings memories of chanting in a classroom.

Affirmative, Indeed

At the command prompt type yes I think Nitrous.IO is AWESOME

Try this too: yes I \<3 Nitrous.IO

Use Ctrl+c to stop (it takes a few seconds to respond)

Tweet It

You can send a quick Tweet from the Command Line like this

url -u TwitterUsername:TwitterPassword -d status=“This Tweet was sent from the command line in a @Nitrous.IO box. Check out Nitrous.tips for details” http://twitter.com/statuses/update.xml
